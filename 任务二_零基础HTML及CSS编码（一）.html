一.CSS简介

CSS 指层叠样式表 (Cascading Style Sheets)
样式定义如何显示 HTML 元素
样式通常存储在样式表中
把样式添加到 HTML 4.0 中，是为了解决内容与表现分离的问题
外部样式表可以极大提高工作效率
外部样式表通常存储在 CSS 文件中
多个样式定义可层叠为一个

二.CSS语法

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。


三.属性选择器

举例说明各种选择器的用法：
格式一：E[attr]{rules} 选择具有attr的E元素，并应用rules指定样式
格式二：E[attr=value]{rules} 选择具有attr属性，并且属性值等于value的E元素，并应用rules指定样式
格式三：E[attr~=value]{rules} 选择具有attr属性并且属性值中包含多个字符列表，其中任意字符等于value的E元素，并应用rules指定样式
格式四：E[attr|=value]{rules} 选择具有attr属性的E元素，并且属性值为用连字符分隔的字符列表，且以value开始的E元素，并应用rules指定样式
格式五：E[attr^=value]{rules} 选择具有attr属性的E元素，并且属性值以value开始的E元素，并应用rules指定样式
格式六：E[attr$=value]{rules} 选择具有attr属性的E元素，并且属性值以value结尾的E元素，并应用rules指定样式
格式七：E[attr*=value]{rules} 选择具有attr属性的E元素，并且属性值为中包含value的E元素，并应用rules指定样式
*[title] {color:red;} /*样式1*/
a[href]{font-size:25px;} /*样式2*/
a[href][id]{background-color:yellow} /*样式3*/
a[href="http://baoquanhu.cn"]{font-weight:bolder} /*样式4*/
span[title~=big]{margin-left:20px;} /*样式5*/
span[title|=big]{margin-left:40px} /*样式6*/

四.类选择器

类选择器允许以一种独立于文档元素的方式来指定样式。
该选择器可以单独使用，也可以与其他元素结合使用。
提示：只有适当地标记文档后，才能使用这些选择器，所以使用这两种选择器通常需要先做一些构想和计划。
要应用样式而不考虑具体设计的元素，最常用的方法就是使用类选择器。

五.伪类选择器

CSS 伪类用于向某些选择器添加特殊的效果。

1.锚伪类

在支持 CSS 的浏览器中，链接的不同状态都可以不同的方式显示，
这些状态包括：活动状态，已被访问状态，未被访问状态，和鼠标悬停状态。
a:link {color: #FF0000} /* 未访问的链接 */
a:visited {color: #00FF00} /* 已访问的链接 */
a:hover {color: #FF00FF} /* 鼠标移动到链接上 */
a:active {color: #0000FF} /* 选定的链接 */

2.first-child,last-child,nth-child,nth-last-child伪类

  • first-child:用于选择某元素的第一个子元素
  • last-child:用于选择某元素的最后一个元素
  • nth-child:用于选择某元素指定正顺序的子元素
  • nth-last-child:用于选择某元素倒序书顺序的元素
例如：
/*对某元素第一个P元素应用样式，文字向右平移20像素*/
p:first-child{margin-left:20px}
/*对某元素最后一个P元素应用样式，字体大小为30像素*/
p:last-child{font-size: 30px;}
/*对某元素正数第3个LI元素应用样式，文字向右平移20像素*/
li:nth-child(3){margin-left:20px}
/*对某元素倒数第2个LI元素应用样式，字体大小为30像素*/
li:nth-last-child(2){font-size:30px;}

3.E:hover,E:active和E:focus选择器

  • E:hover:用于设定当鼠标悬浮于所选择的元素时使用的样式
  • E:active用于设置当鼠标点击时的样式
  • E:focus用于设置元素获取焦点时的样式
例如：
/*当鼠标悬停于div元素上时，为div元素增加边框*/
div:hover{border:1px solid blue;}
/*当鼠标点击div元素时，div元素背景色设置为绿色*/
div:active{background-color:green}
/*当input元素获得焦点时，input元素背景色设置为灰色*/
input:focus{background-color:gray}
